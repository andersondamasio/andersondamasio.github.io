Gerenciando Arquivos no Terminal: Os 5 Melhores Gerenciadores de Arquivos Linux que Você Deveria Conhecer

Publicado em: 25/01/2026 06:24

Se você é um usuário Linux, sabe que em muitas situações é preciso sair da zona de conforto da interface gráfica e ir direto para o terminal. Isso pode parecer um desafio para alguns, mas, na verdade, pode ser uma experiência bastante gratificante e eficiente. Eu mesmo, como arquiteto de software, muitas vezes me vejo em servidores remotos ou até mesmo em máquinas locais, onde a linha de comando se torna uma aliada poderosa. Então, vou compartilhar com vocês cinco gerenciadores de arquivos baseados em terminal que são gratuitos e super fáceis de usar.

Por que usar gerenciadores de arquivos em terminal?

Os gerenciadores de arquivos que operam no terminal podem parecer antiquados à primeira vista, mas eles têm suas vantagens. Por exemplo, você pode gerenciar arquivos de forma mais rápida e eficiente, especialmente quando está conectado. a servidores remotos. Além disso, muitos deles são incrivelmente leves e consomem muito menos recursos do que suas contrapartes gráficas. E quem não ama um pouco de nostalgia com um toque de eficiência?

Conheça os 5 melhores gerenciadores de arquivos para Linux

1. Midnight Commander

O Midnight Commander (ou MC, como é mais conhecido) é o clássico dos clássicos. Ele já está por aí há tanto tempo que eu me lembro de ter usado nas minhas primeiras experiências com Linux. O MC combina uma interface que imita uma GUI com a funcionalidade. do terminal, permitindo até mesmo navegação com o mouse! É impressionante como ele oferece recursos como visualização em duas colunas, suportte a FTP, e até acesso remoto via SSH. Para instalá-lo, basta usar:

  • Ubuntu/Debian: sudo apt-get install mc -y
  • Fedora: sudo dnf install mc -y
  • Arch: sudo pacman -S mc

2. Yazi

Se você busca algo mais moderno, o Yazi pode ser a sua praia. Escrito em Rust, ele é incrivelmente rápido e, o melhor, respeita o tema do seu sistema. Uma das funcionalidades mais legais é a pré-visualização de imagens, mas atenção: isso só funciona em terminais específicos. Se você não estiver usando um deles, as imagens podem ficar uma bagunça. Para instalar, utilize:

flatpak install yazi

3. Ranger

O Ranger é perfeito para quem ama a eficiência do Vim. Com uma interface de três colunas, ele facilita a navegação e a visualização dos arquivos. O que eu gosto é que, ao navegar, ele já abre uma pré-visualização do arquivo selecionado. Mas cuidado, se estiver em um servidor remoto, você pode não conseguir visualizar vídeos e imagens como desejado. A instalação é simples:

  • Debian/Ubuntu: sudo apt-get install ranger -y
  • Fedora: sudo dnf install ranger -y
  • Arch: sudo pacman -S ranger

4. Nnn

Agora, se você é do tipo minimalista, o Nnn é a sua melhor escolha. Com uma interface super clean, ele permite que você navegue apenas usando as setas do teclado. É tão leve que nem dá pra acreditar! Apesar de ser simples, ele vem com um monte de funcionalidades úteis, como filtros e suporte a arquivos compactados. Para instalar, use:

  • Ubuntu/Debian: sudo apt-get install nnn -y
  • Fedora: sudo dnf install nnn -y
  • Arch: sudo pacman -S nnn

5. Fim das apresentações

Esses gerenciadores são apenas algumas opções que podem facilitar a vida de quem precisa trabalhar no terminal. Cada um tem suas particularidades e, dependendo do seu fluxo de trabalho, um pode se destacar mais que o outro. A chave é experimentar e descobrir qual deles se encaixa melhor nas suas necessidades.

Conclusão

Gerenciar arquivos através do terminal pode não ser a escolha mais popular entre os usuários de Linux, mas com as ferramentas certas, pode se tornar uma experiência bastante eficiente. Seja você um desenvolvedor, um administrador de sistemas ou apenas um entusiasta, vale a pena explorar essas opções. No fim das contas, o que importa é encontrar o que funciona para você e deixar sua vida mais prática. E, quem sabe, você não acaba se apaixonando pela linha de comando!
