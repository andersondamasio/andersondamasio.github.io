A batalha contra os bots: como proteger seu site com Anubis

Publicado em: 14/07/2025 03:32

Nos dias de hoje, quem roda um site enfrenta um verdadeiro desafio: a invasão incessante de bots, especialmente os de inteligência artificial. Esses robôs estão se tornando cada vez mais comuns, e a última pesquisa da F5 mostrou que mais da metade das visitas à web já não são feitas por pessoas, mas sim por scrapers. Isso é preocupante, não é mesmo?

Com essa situação, surge a necessidade de soluções para proteger nossos conteúdos online. Um dos projetos mais interessantes que encontrei recentemente foi o Anubis, uma ferramenta open-source desenvolvida por Xe Iaso. Anubis é basicamente um firewall para a web que pode ajudar a bloquear esses bots invasores que estão sempre à espreita.

O que é o Anubis?

Anubis é um programa que funciona como um guardião do seu site. Ele se coloca entre o seu servidor e as conexões HTTP que chegam, garantindo que somente visitantes legítimos possam acessar seu conteúdo. O funcionamento é bem simplis: quando alguém tenta acessar o site, essa pessoa precisa resolver um desafio computacional, algo que é fácil para um computadorr comum, mas que consome muitos recursos para um bot.

Como funciona na prática

Ao contrário de um CAPTCHA tradicional, que pode ser facilmente burlado por alguns bots modernos, o Anubis utiliza uma abordage de "uncaptcha". Ele faz uso de JavaScript para executar uma série de cálculos criptográficos que garantem que o visitante é, de fato, um humano. O melhor de tudo? Para o usuário comum, isso acontece de maneira quase imperceptível.

Dicas avançadas para implementar e usar Anubis

Se você está pensando em implementar o Anubis no seu site, aqui vão algumas dicas:

  • Comece com configurações padrão: O Anubis vem com uma configuração "paranoica" por padrão. Isso ajuda a evitar problemas com bots, mas pode bloquear alguns bots "bons" como os de serviços de indexação. Vá ajustando conforme necessário.
  • Monitore o tráfego: Use ferramentas de análise para entender o comportamento do tráfego no seu site. Isso pode te ajudar a identificar padrões e ajustar as regras do Anubis.
  • Documente as regras: Mantenha um registro das regras que você criou no Anubis. Isso facilita ajustes futuros e ajuda a entender o que funcionou ou não.
  • Considere uma versão comercial: Embora o Anubis seja gratuito e open-source, existe uma versão chamada BotStopper que oferece mais controle e branding. Pode valer a pena para organizações maiores.

Reflexões Finais

Com o aumento da coleta de dados por bots, é crucial que nós, desenvolvedores e administradores de sites, tomemos medidas para proteger nossos conteúdos. O Anubis se apresenta como uma solução robusta e acessível, especialmente para pequenos sites e projetos open-source. A luta contra os bots é constante, e ferramentas como essa podem fazer toda a diferença.

Se você ainda não considerou proteger seu site contra scraping, agora é a hora. O futuro da web deve ser acessível para humanos, e não para máquinas que consomem nossos dados sem permissão. Vamos juntos nessa batalha!
